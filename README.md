# API-Plataforma-Blogs

Cree una API RESTful para una plataforma de blogs personal

# RoadMap Web

Objetivos
Los objetivos de este proyecto son ayudarte:

Comprenda qué son las API RESTful que incluyen las mejores prácticas y convenciones
Aprende a crear una API RESTful
Aprenda sobre métodos HTTP comunes como GET, POST, PUT, PATCH, DELETE
Aprenda sobre códigos de estado y manejo de errores en API
Aprenda a realizar operaciones CRUD utilizando una API
Aprende a trabajar con bases de datos
Requisitos
Debe crear una API RESTful para una plataforma de blogs personal. La API debe permitir a los usuarios realizar las siguientes operaciones:

Crea una nueva publicación de blog
Actualizar una publicación de blog existente
Eliminar una publicación de blog existente
Consigue una sola publicación de blog
Consigue todas las publicaciones del blog
Filtrar publicaciones de blog por un término de búsqueda
